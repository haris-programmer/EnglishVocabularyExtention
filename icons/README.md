# Extension Icons

This folder should contain the extension icons in PNG format:

- `icon-16.png` - 16x16 pixels (toolbar icon)
- `icon-32.png` - 32x32 pixels (toolbar icon)
- `icon-48.png` - 48x48 pixels (extension management page)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create your own icons or use these suggested specifications:

- **Style**: Modern, clean design with a book or language learning theme
- **Colors**: Use the extension's brand colors (#667eea, #764ba2)
- **Format**: PNG with transparent background
- **Content**: Consider using symbols like üìñ, üî§, or üìù

## Temporary Solution

For development/testing, you can:
1. Create simple colored squares as placeholders
2. Use online icon generators
3. Design custom icons using tools like Figma, Canva, or Photoshop

The extension will work without icons, but Chrome will show default placeholder icons.
