## Quick Start

### Step 1: Load the Extension in Chrome

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **"Load unpacked"**
4. Select the `english-helper-extension` folder
5. The extension should now appear in your extensions list

### Step 2: Configure Your OpenAI API Key

1. Click the English Helper extension icon in your Chrome toolbar
2. Click the settings (‚öôÔ∏è) button in the popup
3. Enter your OpenAI API key:
   - Go to [OpenAI Platform](https://platform.openai.com/api-keys)
   - Sign in and create a new API key
   - Copy and paste it into the extension settings
4. Choose your preferred AI model (GPT-3.5 Turbo recommended for cost)
5. Click **"Save Settings"**

### Step 3: Test the Extension

1. Go to any website with English text
2. Select (highlight) a word or phrase
3. Right-click and choose **"Explain with English Helper"**
4. View the comprehensive explanation in the popup

## Troubleshooting

### Extension not loading?
- Make sure Developer mode is enabled
- Check that you selected the correct folder
- Look for any error messages in the Extensions page

### API key not working?
- Verify the key starts with "sk-"
- Check your OpenAI account has credits
- Make sure you copied the full key without spaces

### Text selection not detected?
- Ensure text is properly highlighted
- Try refreshing the page
- Check that the website allows content scripts

## Usage Tips

- **Best Results**: Select complete words or short phrases (avoid partial words)
- **Cost Management**: Use GPT-3.5 Turbo for everyday use, GPT-4 for complex terms
- **Learning**: Use the highlight feature to remember words you've looked up
